说明：
Core：STM32的核心源文件和头文件；
DSP_Lib：STM32下DSP的库文件，里面有FFT函数；
Hardware：STM32的外设和外围电路的初始化程序和相关底层程序；
Output：工程的输出文件夹，内含template.hex；
ProcessingData：STM32测量电压、电流的数据处理程序，主要是VA档和W档；
STM32F37X_StdPeriph_Driver：STM32的外设的源文件和头文件；
System：中断初始化、延时、低功耗程序；
User：用户主程序main.c、按键和显示等用户交互程序userVAinteraction.c、中断程序stm32f37x_it.c等，工程文件template.uvproj位于User文件夹下。

各硬件的调用函数见相应的底层文件！！！

			PreemptionPriority		SubPriority

DMA2		0						1				SDADC数据DMA中断
Usart1		0						2				DTA0660通信串口中断，用于单片机与DTA通信
USART2		0						3				与上位机通信串口中断


TIM5		1						0				比较器捕获中断
TIM12		1						1				1s中断：秒表显示闪烁
TIM14		1						2				0.5s中断：相序测量闪烁
TIM4		1						3				蜂鸣器频率中断

EXTI9_5		2						0				按键外部中断

TIM13		3						3				1s中断，计数1800次为30min，进入Standby模式
